# Smart Parking Mobile

Мобильное приложение на React Native / Expo для базовых сценариев из НТО:

- вход и регистрация арендатора или владельца парковочного места;
- просмотр парковок и свободных мест;
- выбор места на цифровой карте;
- создание, оплата и отмена бронирования;
- регистрация автомобиля;
- базовый кабинет арендодателя: статус мест, ставка, доход;
- задел под уведомления, штрафы и маршрут до места.

По умолчанию приложение подключается к реальному API через `api-gateway` на порту `8080`.
Demo-режим включается только явно: `EXPO_PUBLIC_DEMO_MODE=true`.

## Запуск с backend

```bash
cd ..
docker compose up --build
cd mobile
npm install
npx expo start -c
```

После запуска:

1. Установите Expo Go на телефон или откройте Android/iOS эмулятор.
2. Отсканируйте QR-код из терминала.
3. Зарегистрируйте пользователя или войдите существующей учетной записью.

Приложение автоматически берет IP из Expo URL и подключается к `http://<ip-компьютера>:8080`.
Если автоопределение не сработало, задайте адрес вручную:

```powershell
$env:EXPO_PUBLIC_API_URL="http://192.168.1.10:8080"
npx expo start -c
```

Для Android Emulator обычно нужен адрес хоста:

```powershell
$env:EXPO_PUBLIC_API_URL="http://10.0.2.2:8080"
npx expo start -c
```

## Demo-режим без backend

```powershell
cd mobile
$env:EXPO_PUBLIC_DEMO_MODE="true"
npx expo start -c
```

## Что проверять вручную

1. Вход в demo-режиме открывает главный экран.
2. На вкладке "Парковки" выбирается парковка и свободное место.
3. Кнопка бронирования создает активную бронь и переводит на вкладку "Бронь".
4. На вкладке "Бронь" работают оплата и отмена.
5. На вкладке "Авто" добавляется номер автомобиля.
6. На вкладке "Места" меняется статус мест владельца.
7. На вкладке "Профиль" отображается текущий API URL.
